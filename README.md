## 📂 Featured Projects

### 🏠 [House Price Prediction — Regression Pipeline](https://github.com/pmash634/house-price-prediction)
> *Python · XGBoost · Scikit-learn · Pandas · Feature Engineering*

End-to-end ML pipeline predicting residential property prices on Kaggle's real-estate dataset.

- ✅ Engineered **neighbourhood-level aggregate features** (mean/median price per zone) to capture spatial pricing signals
- ✅ Applied **log transformation** to correct target skewness and reduce prediction error
- ✅ Benchmarked **Random Forest vs XGBoost** via 5-fold cross-validation
- ✅ Optimised with **grid-search hyperparameter tuning**
- ✅ Fully reproducible pipeline from raw data to evaluated model

### 📉 [Dimensionality Reduction with PCA](https://github.com/pmash634/pca-dimensionality-reduction)
> *Python · Scikit-learn · Matplotlib · NumPy*

Explored Principal Component Analysis to compress high-dimensional feature spaces without sacrificing predictive power.

- ✅ Retained **95% of explained variance** with significantly fewer features
- ✅ Visualised scree plots and **component loadings** to interpret feature contributions
- ✅ Validated that downstream model performance was preserved post-reduction
